Varibles is Javascript

Variables are containers to store data values

There are three types of variables in Javascript

  1. Var
  2. let
  3. const

Rules for defining a variables

  1. Variables are case sensitive
  2. can be started with a underscore ( _ ) or dollar ($)
  3. Cannot start with a number
  4. we can use letters and numbers
  5. Small and capital alphabets are allowed
  6. Javascript reserved key words are not allowed

Rules for var, let and const

  1. Var ,let and const can be used to store the values.
  2. var is global scoped.
  3. let and const are block scoped.
  4. var can be updated and re-declared.
  5. let can be updated but not re-declared.
  6. const and neither be update nor re-declared.
  7. always try to use let or const.

Cases to write variables in javascript

We have four cases to write variables in javascript

  1. camelCase
  2. kebab-case
  3. snake_case
  4. PascalCase

Always try to prefer camel case
