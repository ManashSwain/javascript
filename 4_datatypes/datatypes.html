Primitive and reference data type in javascript

There are two types of datatypes in javascript.They are :

  1. Primitive data type
  2. Reference or non primitive data type

Primitive data types in javascript

There are 7 types of primitive data type . They are

  1. Number
  2. String
  3. Undefined
  4. Null
  5. Boolean
  6. Symbol
  7. BigInt

Reference or non primitive data types in javascript

There are 4 types of non primitive data types in javascript . They are

  1. Arrays
  2. Object Literals
  3. Functions
  4. Date objects
